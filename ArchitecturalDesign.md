
# Technology Used

- ASP.NET Core Blazor Web App
- SQLite
- SQLAlchemy
- FastAPI

## Interactions

- SQLite database is implemented using the StudentTracker.db file.
- The back-end API uses FastAPI and SQLAlchemy to interact with the database file.
- The front-end API uses C# files to provide helper services for site functionality.

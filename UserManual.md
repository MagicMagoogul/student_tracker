# User's Manual

## Home Page

### For all users

This page introduces users to the website and directs them to log in as needed.
The navigation bar allows you to navigate to all other pages on the site.
As of the current release, the pages are viewable by all, but will be locked down based on user type in future releases.

## Login Page

### For all users

To log in:

- Enter your email in the first text box
- Enter your password in the second box
- Press submit
- If you have correctly logged in, you will see a success message and will be greeted by your first name.

## Log Hours Page

### For students

To log your hours:

- Enter the number of hours you worked, the date that you worked them, and the location of your work in the corresponding text boxes
- Press submit
- When the success message comes on screen, you have successfully logged your Hours

## View Student Records Page

### For professors

This page shows all students, their ENumbers, and the number of hours that they have completed.

## Messaging Page

### For students and professors

As of now, this page is just a template for future messaging functionality.
The page shows a text box where the user can enter a message and then select which individual to send it to.

## Admin Manage Users Page

### For admins

To create a user:

- Select the user type to be added (student, professor, or admin)
- Press create
- Enter the required information in the corresponding text boxes at the center of the page
- Press submit

To delete a user:
- Select the user type to be deleted
- Enter the user ID of the user you wish to delete
- If you do not know the user ID, enter any user information you know in the searching section to search for your user
- Press delete

## Admin Manage Database Page

### For admins

To modify information about a user:

- Enter the user ID of the user you wish to modify
- If you do not know the user ID, enter any user information you know in the searching section to search for your user
- Select load
- Change the desired information in the text boxes that come up
- Select modify

## Additional Notes

Admins will be able to view all pages.
